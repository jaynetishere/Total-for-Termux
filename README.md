# Total is a **CLI-based real-time chatting platform** designed primarily for Linux environments.
[This version is for Termux NOT linux]
Experience fast, terminal-native communication directly from your command line.

---

## Features
* **CLI first:** Degine natively for Command-Line-Interface.
* **Linux First:** Degine to work for Linux first.
* **Real-time:** Instant messaging capabilities.
* **Lightweight:** Built for speed and efficiency.

---

## InstallationYou must have **Go** and **Git** installed on your system to install Total.

### Termux Run the following command in your terminal:

```bash
curl -fsSL https://raw.githubusercontent.com/jaynetishere/Total-for-Termux/main/Installation/install.sh | bash
```
